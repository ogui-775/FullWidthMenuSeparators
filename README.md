# FullWidthMenuSeparators

What does this do? Makes all system context menus (dock, menu bar, right-click) have full-width menu separators, reminiscent of older styles. What does it not do? Literally anything else.

1 ) Compile source for target architecture. For ARM, use arm64 & arm64e.

2 ) Drag created dylib to /private/var/ammonia/core/tweaks/.

3 ) Create a compatible blacklist file.

4 ) Restart.

OR 

1 ) Download precompiled dylib and blacklist.

2 ) Move both to /private/var/ammonia/core/tweaks/.

3 ) Restart.
